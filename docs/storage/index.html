Global Storage

Global storage is used to store:

  • artifacts that are created during job execution (used by artifacts tool)
  • Git repository bundles to speed up cloning repositories (used by git tool)
  • caches of files e.g. npm node_modules to speed up their retrieval (used by cache tool)

Storage is backed by Minio, an AWS S3 compatible object storage.
