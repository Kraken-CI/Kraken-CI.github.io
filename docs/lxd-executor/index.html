LXD Executor

Images​

LXD images are hosted on https://images.linuxcontainers.org/. Example name: 'ubuntu/focal/amd64'. To get list of all available LXD container names run lxc image list images: command.

The list of the most popular images:

ImageDescription
ubuntu/focal/amd64Ubuntu Focal 20.04
ubuntu/bionic/amd64Ubuntu Bionic 18.04
fedora/33/amd64Fedora 33
centos/8/amd64CentOS 8
debian/buster/amd64Debian Buster 10
debian/bullseye/amd64Debian Bullseye next
alpine/3.13/amd64Alpine 3.13
opensuse/15.3/amd64openSUSE Leap 15.3
