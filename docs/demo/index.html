Demo

It is possible to check Kraken CI just now. Either visiting online demo or running it under the desk.

Online#

Online demo version can be visited on: http://lab.kraken.ci/

Under your desk#

To start demo instance run:

rake docker_up

Then open http://localhost:8080/
